# Guide for the Day

You can follow the below guide, which includes plenty of prompts and ideas, as well as handy references. 

## Building a simple HTML page

Without planning:
```
Create a file called index.html with a page that says Good Morning and shows today’s date.
Make the page fun with some interactive features.
```
With planning:
```
I want to plan a homepage index.html that says Good Morning and shows today’s date.
Make the page fun with some interactive features.
```
## Building a simple next.js landing page
```
Create a Next.js app with a landing page, nav bar, and card layout.
Make the theme all about [how teachers can save time with AI.
Bring in some serious and not so serious ideas.]
Make the landing page look like something build by Microsoft Frontpage  in the 1990s.
```
